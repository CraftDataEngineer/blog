Description

Le Data Engineer a pour mission d’aider sa structure à exploiter ses données et proposer des services pertinents à ses clients ! Il doit être en mesure de développer du code pour lire, transformer et stocker la donnée au bon format.

Lors de cette formation, nous allons :

  • Apprendre à manipuler les données avec Python et SQL.
  • Utiliser les bonnes pratiques de code pour développer proprement.

Objectifs

  • Apprendre à développer avec Python.
  • Requêter les données avec le langage SQL.
  • Maîtriser les commandes de bases de Linux.
  • Utiliser Git pour collaborer librement sur des projets de code.

Public Cible

  • Tout le monde

Prérequis

  • Des connaissances de bases en informatique.
